# GlowCast

macOS menu-bar app to control HyperX QuadCast 2 S RGB lighting (solid color,
brightness, animations). Drives the mic continuously via IOHIDManager.

Protocol reverse-engineered with help from https://github.com/Ors1mer/QuadcastRGB (GPL-2.0).

## Build
    bash Scripts/make_app.sh
    open build/GlowCast.app
